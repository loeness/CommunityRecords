# Members

This directory contains a personal learning space for each community member.

---

## How to Join

1. Follow the steps in [CONTRIBUTING.md](../CONTRIBUTING.md) to fork and clone the repository.
2. Create a folder under `members/` using your GitHub username as the folder name:
   ```bash
   mkdir -p members/<your-username>/logs
   ```
3. Copy the member profile template and fill it in:
   ```bash
   cp templates/member-profile.md members/<your-username>/profile.md
   ```
4. Open a Pull Request with your profile. Once it is merged, you are officially a member.

---

## Directory Layout per Member

```
members/
└── <your-username>/
    ├── profile.md     # Who you are, your background, and your learning goals
    └── logs/          # One file per day
        └── YYYY-MM-DD.md
```
